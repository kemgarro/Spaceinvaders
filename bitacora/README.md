# Bitacora del proyecto

Registro diario de actividades del equipo. Una entrada por dia con formato `YYYY-MM-DD.md`.

## Formato sugerido
```markdown
# Bitacora — DD de mes de YYYY

## Participantes
- Persona A, Persona B, Persona C

## Actividades realizadas
- [Persona A] ...
- [Persona B] ...

## Decisiones tomadas
- ...

## Problemas encontrados
- ...

## Siguiente sesion
- ...
```

## Recordatorio
La bitacora forma parte de los entregables obligatorios (RNF-17 del enunciado).
